About

Rocket Request is a utility tool for the factory automation game, Factorio.
It shows you how to distribute items across your rocket silos, minimizing the number of required launches.
A first-fit-decreasing algorithm is utilized to calculate this distribution.

Why?

When auto-requesting to a space platform, silos always fill up to capacity with a single item, leading to wasteful launches.
Of course, this is not a concern in a game where resources are nearly unlimited; however, wasting resources bothers me.
So instead, I wasted my time to create a program, and then I turned it into a web application for fun.

Source

If you would like to check out or contribute to the source code, it is available on GitHub at brufinus/rocket-request.
