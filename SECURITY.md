# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

Click [here](https://github.com/brufinus/rocket-request/security/advisories/new) to report a vulnerability,
or go to the [Security and quality](https://github.com/brufinus/rocket-request/security) tab.

Vulnerabilities will be resolved as soon as possible.
