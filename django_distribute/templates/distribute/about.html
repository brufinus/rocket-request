About

Rocket Request is a utility tool for the factory automation game, Factorio.
It shows how to distribute items across rocket silos, aiming to minimize the number of required launches.
A first-fit-decreasing algorithm is utilized to calculate this distribution.

Why?

When auto-requesting to a space platform, silos always fill up to max capacity on a single item, leading to wasteful launches.
Of course, this is not a concern in a game where resources are nearly unlimited; however, it could be helpful in the early game.
Thus, the intended use of this tool is to show you how to distribute a small number of items amongst a small number of silos, so that you can distribute and launch those items manually in-game.

Blueprint Import/Export

The import functionality only allows for the import of vanilla items from a blueprint. The field also has a maximum length, since it's not intended for the user to distribute an inordinate amount of items.

The resulting blueprint string is a book that contains blueprints of requester chests. Each chest corresponds to a silo and requests the items distributed to that silo. The intended use of this book is to place each chest next to its silo so that its contents can be inserted into the silo automatically with an inserter.

Source

If you would like to check out or contribute to the source code, it is available on GitHub at brufinus/rocket-request.
